HOW TO SET UP THIS STRUCTURE
────────────────────────────

1. Create root folder:
   mkdir review-sheet-generator
   cd review-sheet-generator

2. Create app directory structure:
   mkdir -p app/api/generate

3. Place files in correct locations:
   - route.js → app/api/generate/
   - page.js → app/
   - layout.js → app/
   - globals.css → app/
   - page.module.css → app/
   - package.json → root
   - next.config.js → root
   - vercel.json → root
   - .env.example → root
   - .gitignore → root
   - All .md files → root

4. Install dependencies:
   npm install

5. Run locally:
   npm run dev

6. Deploy to Vercel:
   See QUICKSTART.md or DEPLOYMENT.md

ENVIRONMENT VARIABLES
─────────────────────

Local Development (.env.local):
  ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxx

Vercel Production (Settings → Environment Variables):
  ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxx

Available in Code:
  - process.env.ANTHROPIC_API_KEY (server-side only)

  - process.env.ANTHROPIC_API_KEY (client-side)
