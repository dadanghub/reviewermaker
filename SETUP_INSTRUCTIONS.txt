OVERVIEW
--------
This is a professional, educator-focused web application that generates 
curriculum-aligned review sheets in seconds using Claude AI.

Website Features:
  ✓ Professional review sheet generation
  ✓ Curriculum-aligned content (Grade 4-6)
  ✓ Print & download functionality
  ✓ Responsive design (mobile, tablet, desktop)
  ✓ Fast deployment on Vercel (free tier available)

═══════════════════════════════════════════════════════════════════════════════

STEP 1: PREPARE YOUR API KEY (2 minutes)
────────────────────────────────────────

1. Go to: https://console.anthropic.com
2. Sign up or log in
3. Navigate to: Account → API Keys
4. Click: "Create Key"
5. Copy your API key (you'll need this soon!)
6. Keep it secure - don't share it

STEP 3: DEPLOY TO VERCEL (3 minutes)
────────────────────────────────────

OPTION A: GitHub + Vercel (Recommended)
────────────────────────────────────────

1. Create GitHub account at https://github.com (if you don't have one)

2. Create new repository:
   - Go to https://github.com/new
   - Name: "review-sheet-generator"
   - Public (recommended for learning)
   - Click "Create repository"

3. Push code to GitHub:
   ```bash
   cd review-sheet-generator
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/review-sheet-generator.git
   git branch -M main
   git push -u origin main
   ```

4. Deploy to Vercel:
   - Go to https://vercel.com/new
   - Click "Import Git Repository"
   - Paste your GitHub repo URL
   - Select your GitHub repo when it appears
   - Click "Import"

5. Configure Vercel:
   - Framework: "Next.js" (should auto-detect)
   - Root Directory: "./" (default)
   - Build Command: npm run build (default)
   
   ENVIRONMENT VARIABLES:
   ┌─────────────────────────────────────┐
   │ Name:  ANTHROPIC_API_KEY            │
   │ Value: [paste your API key here]    │
   └─────────────────────────────────────┘
   
   Click "Deploy"

6. Wait 2-3 minutes, then Vercel will show your live URL! 🎉

OPTION B: Direct Vercel Deployment (If no GitHub)
───────────────────────────────────────────────────

1. Go to https://vercel.com
2. Sign up with Google or GitHub
3. Click "Create"
4. Select "Other" or "Next.js"
5. Drag & drop your project folder OR manually upload files
6. Add environment variable:
   - ANTHROPIC_API_KEY: [paste your key]
7. Click "Deploy"

OPTION C: Vercel CLI (If you prefer command line)
──────────────────────────────────────────────────

1. Install Vercel CLI:
   npm install -g vercel

2. Deploy:
   vercel

3. Follow prompts:
   - Log in to your account
   - Link project
   - Select framework (Next.js)

4. Add environment variable:
   vercel env add ANTHROPIC_API_KEY
   [paste your API key when prompted]

5. Redeploy with environment:
   vercel --prod

TROUBLESHOOTING
───────────────

Problem: "Failed to generate review sheet"
Solution: 
  1. Go to Vercel project settings
  2. Check Environment Variables
  3. Verify ANTHROPIC_API_KEY is correct
  4. Redeploy

Problem: API key error
Solution:
  1. Generate new API key at https://console.anthropic.com
  2. Update in Vercel settings
  3. Redeploy

Problem: Page won't load
Solution:
  1. Check internet connection
  2. Clear browser cache (Ctrl+Shift+Delete)
  3. Try different browser
  4. Check Vercel deployment status

Problem: Review generation times out
Solution:
  1. This is normal - reviews take 20-30 seconds
  2. Don't close the browser tab
  3. Check internet speed
  4. Try with fewer topics

CUSTOMIZATION
─────────────

Change Grade Level:
  Edit app/api/generate/route.js
  Find "Grade 4–6" and change to desired grade

Change Colors:
  Edit app/globals.css
  Modify --primary, --accent variables

Add Your School Logo:
  1. Put logo in public/ folder
  2. Add <img> tag in app/page.js

Change Target Audience:
  Edit the systemPrompt in app/api/generate/route.js
